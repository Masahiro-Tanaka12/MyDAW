## スケール定義

### メジャースケール（長調）

```
Cメジャー: C - D - E - F - G - A - B
Gメジャー: G - A - B - C - D - E - F#
Fメジャー: F - G - A - Bb - C - D - E
```

ダイアトニックコード（Cメジャー）：

| 度数 | コード | 種類 | 使いやすさ |
|---|---|---|---|
| I | C | メジャー | ★★★ |
| ii | Dm | マイナー | ★★ |
| iii | Em | マイナー | ★★ |
| IV | F | メジャー | ★★★ |
| V | G | メジャー | ★★★ |
| vi | Am | マイナー | ★★★ |
| vii | Bdim | ディミニッシュ | ★（使わない） |

### マイナースケール（短調）

```
Aマイナー: A - B - C - D - E - F - G
Eマイナー: E - F# - G - A - B - C - D
```

ダイアトニックコード（Aマイナー）：

| 度数 | コード | 種類 |
|---|---|---|
| i | Am | マイナー |
| III | C | メジャー |
| IV | Dm | マイナー |
| V | Em / E | マイナー / メジャー |
| VI | F | メジャー |
| VII | G | メジャー |

### ペンタトニックスケール（ゲーム音楽向け）

```
Cペンタトニック（メジャー）: C - D - E - G - A
Aペンタトニック（マイナー）: A - C - D - E - G
```

## メロディパターン (melody-patterns.json)

メロディは **スケール上の相対音程** で定義する。実装時にキーに応じて移調される。

### シンプルステップ
- **ID**: `mp-001`
- **表示名**: やさしいメロディ
- **説明**: 音が隣り合う音へ順番に動く。覚えやすい。
- **音の動き**: ↑ ↑ → ↑ ↓ ↓ → ↓（ステップワイズ）
- **音域**: 1オクターブ以内
- **対応雰囲気**: happy, relaxed

### 跳躍メロディ
- **ID**: `mp-002`
- **表示名**: 元気なメロディ
- **説明**: 音が大きくジャンプする。印象的でエネルギッシュ。
- **音の動き**: 1度 → 5度 → 3度 → 8度（跳躍）
- **音域**: 1オクターブ半
- **対応雰囲気**: happy, epic

### 波形メロディ
- **ID**: `mp-003`
- **表示名**: 流れるメロディ
- **説明**: 上がって下がる山型の動き。バラードに合う。
- **音の動き**: ↑ ↑ ↑ ↓ ↓ ↓（山型）
- **音域**: 1オクターブ
- **対応雰囲気**: sad, relaxed

### リズミカル
- **ID**: `mp-004`
- **表示名**: ノリのいいメロディ
- **説明**: リズムが特徴的。裏拍を使って躍動感を出す。
- **音の動き**: 休符 ♩ ♩ ♩（シンコペーション）
- **音域**: 1オクターブ以内
- **対応雰囲気**: cool, happy

### 繰り返しリフ
- **ID**: `mp-005`
- **表示名**: 耳に残るメロディ
- **説明**: 短いフレーズをくり返す。ゲーム音楽向き。
