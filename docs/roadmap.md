# ロードマップ

## 判断基準（全フェーズ共通）

> 「音楽経験ゼロのユーザーが、この機能を見て迷わないか？」
> YES なら実装。NO なら実装しない。

---

## v0.1.0 ✅ 完成（現在）

**テーマ：音が鳴る・作品を持てる**

- [x] ムード選択（元気 / 夜 / 雨 / 春 / おまかせ）
- [x] コード進行・ベース・ドラムの自動生成
- [x] 重み付きランダム抽選による毎回異なる曲
- [x] フェードイン・フェードアウト再生
- [x] マスターコンプレッサー + リミッター
- [x] 曲の保存（LocalStorage）
- [x] マイソング一覧（再生・タイトル編集・削除）
- [x] Mood 別アルバムアートカラー（プレースホルダー）

## v0.3.0 — カスタマイズ（ステップアップ）

**テーマ：慣れてきたユーザーに少しだけ自由を渡す**

- [ ] テンポ（BPM）の手動調整スライダー
- [ ] コード進行の選択（候補3つから選ぶ）
- [ ] 楽器プリセットの選択（音色を変える）

ユーザーが迷わないよう、選択肢は最大3〜4つに絞る。  
設定画面ではなく「選ぶカード」として提示する。

## 実装しない機能（全バージョン共通）

| 機能 | 理由 |
|---|---|
| VST / プラグイン管理 | 初心者には不要、設定コストが高い |
| MIDI 詳細編集 | 専門知識が必要 |
| ピアノロール | 専門的すぎる |
| 複数トラックの手動ミキサー | 自動で十分 |
| テンポ変動（ritardando 等）| 初心者には高度 |
| AI による曲生成 | データベース駆動の方針を維持する |
