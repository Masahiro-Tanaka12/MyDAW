# 🌸 春の曲の音楽知識

## 雰囲気

爽やかで・柔らかく・希望に満ちた印象を与える曲。
桜・卒業・出会い・新しいスタートを連想させる明るさ。
春の風景動画・卒業式BGM・新生活をテーマにした動画に合う。

---

## おすすめ BPM 帯

| テンポ感 | BPM |
|---|---|
| ゆったり春 | 88〜100 |
| 春らしいテンポ | 100〜115 |
| 弾む春 | 115〜124 |

基準：**BPM 104〜112** が「春」の最も雰囲気が出るテンポ帯。
元気よりやや遅く、夜より速い「ちょうどいい軽やかさ」がポイント。

---

## おすすめキー

- **Gメジャー**：「春」の定番キー。温かく・広がりがある
- **Dメジャー**：明るくキラキラした印象
- **Eメジャー**：少し眩しいくらいの明るさ

---

## おすすめコード進行

### ① I → V → vi → IV（王道進行）
```
G → D → Em → C
```
春の曲で最もよく使われる進行。J-POPの春ソングに頻出。
「安心感のある明るさ」が春にぴったり。

### ② I → ii → IV → V
```
G → Am → C → D
```
上昇感があり、期待や希望の感情を表現しやすい。
卒業・出会いのシーンに合う。

### ③ I → VI → IV → V（少し感傷的な春）
```
G → Em → C → D
```
明るさの中に少し切なさが混じる進行。
「出会いと別れ」を同時に感じさせる春らしさがある。

---

## おすすめベースの特徴

- **中程度の長さ**：各コード1小節でルート音を弾く
- **中間のオクターブ**：G2〜D2 の温かみある音域
- **軽やかな音量**：velocity 0.75〜0.85 でやや控えめに

### ベースラインの例（G-D-Em-C に対して）
```
G2（1小節）→ D2（1小節）→ E2（1小節）→ C2（1小節）
```

---

## おすすめドラムパターン

### 4つ打ち（中程度の強さ）
```
キック：1拍・2拍・3拍・4拍
```
元気ほど強くなく、夜ほど弱くない「中間」の強さがポイント。

### 強弱のコツ
- 1拍目・3拍目：少し強め（velocity 0.9）
- 2拍目・4拍目：中程度（velocity 0.8）

### より春らしくするなら（Phase5以降）
- ハイハットを8分音符で刻む（軽やかさが増す）
- スネアは2拍・4拍（クラップのような明るい音）

---

## 避けたい特徴

| 避けるもの | 理由 |
|---|---|
| マイナーキーを主体にする | 暗くなりすぎて春らしさが消える |
| BPM 85 以下 | 重くなりすぎる |
| BPM 130 以上 | 元気になりすぎて春の柔らかさが消える |
| 重たいキック（velocity 最大） | 疾走感が強すぎる |
| 動かないベース（全音符のみ） | 軽やかさが失われる |
