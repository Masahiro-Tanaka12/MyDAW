# 😊 元気な曲の音楽知識

## 雰囲気

明るく・弾むような・前向きな印象を与える曲。
聴いた人が自然と体を動かしたくなる、エネルギッシュな雰囲気。
YouTube BGM・スポーツシーン・日常のポジティブな動画に合う。

---

## おすすめ BPM 帯

| テンポ感 | BPM |
|---|---|
| 少しだけ元気 | 110〜120 |
| 元気 | 120〜132 |
| とても元気 | 132〜148 |

基準：**BPM 128** がポップスで最もよく使われる「元気」の中心値。

---

## おすすめキー

- **Cメジャー**（ド・レ・ミ…）：明るさの王道。初心者が最初に学ぶキー
- **Gメジャー**：Cより少し広がりがあり、爽やかな印象
- **Fメジャー**：温かみがある元気さ

---

## おすすめコード進行

### ① I → V → vi → IV（王道進行）
```
C → G → Am → F
```
世界中のポップスで使われる最も定番の進行。
「安心感」と「盛り上がり」を両立する。

### ② I → IV → V → I（スリーコード）
```
C → F → G → C
```
3つのコードだけで作れるシンプルな進行。
ロック・フォークで特に効果的。

### ③ I → IV → I → V
```
C → F → C → G
```
繰り返しやすく、明るくループしやすい。

---

## おすすめベースの特徴

- **ルート音を打つ**：各コードの根音（C→C、G→G）を1拍目に置く
- **リズムを刻む**：4分音符または8分音符でリズミカルに動かす
- **高めのオクターブ**：C2〜G3 あたりが元気な印象に合う

### ベースラインの例（C-G-Am-F に対して）
```
C2（1小節）→ G2（1小節）→ A2（1小節）→ F2（1小節）
```

---

## おすすめドラムパターン

### 4つ打ち（最もシンプルで元気）
```
キック：1拍・2拍・3拍・4拍すべてに入れる
```
ダンスミュージック・ポップスで最も多用されるリズム。
疾走感が出て「元気」の雰囲気に最も合う。

### 強弱のコツ
- 1拍目・3拍目：強め（velocity 0.9）
- 2拍目・4拍目：少し弱め（velocity 0.8）

---

## 避けたい特徴

| 避けるもの | 理由 |
|---|---|
| マイナーキー（短調） | 暗い印象になる |
| BPM 90 以下 | 重く・暗くなる |
| 長い音符ばかりのベース | リズム感が失われる |
| キックが少ないドラム | 推進力が落ちる |
| 不協和音を含むコード | 不安感が出る |
