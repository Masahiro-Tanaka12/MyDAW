# コード進行 辞書

初心者向けに、よく使われるコード進行を整理したドキュメントです。
「コード」や「進行」という言葉は、ここでは **「曲の流れのパターン」** と読み替えてください。

---

## コード進行とは？

曲の中で「どのコード（和音）をどの順番で鳴らすか」のパターンのことです。
料理のレシピのように、同じ材料（コード）でも順番が変わると全然違う味（雰囲気）になります。

このドキュメントでは「Cの調（ドレミファソラシで始まる）」を例に説明しています。

---

## ① 王道進行（I → V → vi → IV）

```
C → G → Am → F
```

### 雰囲気
明るく・安心感があり・聴きやすい。「これぞポップス」という感じ。

### 使われる場面
世界中のヒット曲の大半がこの進行を使っています。
知らず知らずのうちに何千回も聴いているはずです。

### 特徴
- 最後のFで「また始まる」という感覚が生まれ、自然にループする
- マイナーコード（Am）が1つ混じることで甘い切なさが出る

### おすすめの雰囲気
😊 元気 / 🌸 春 / 明るいバラード

---

## ② カノン進行（I → V → vi → iii → IV → I → IV → V）

```
C → G → Am → Em → F → C → F → G
```

### 雰囲気
荘厳で・感動的・クラシカルな広がり。

### 使われる場面
パッヘルベルの「カノン」という曲が元ネタ。
感動的なCMや映画音楽によく使われます。

### 特徴
- コードが8つあるため、長く聴いても飽きない
- ベースが「ド・ソ・ラ・ミ・ファ・ド・ファ・ソ」と段階的に下がる動きが特徴

### おすすめの雰囲気
🌸 春 / 感動シーン / 卒業・旅立ち

---

## ③ 4536進行（IV → V → iii → vi）

```
F → G → Em → Am
```

### 雰囲気
キラキラした・ドラマチック・盛り上がる感じ。

### 使われる場面
90年代〜2000年代のJ-POPでよく使われました。
サビの盛り上がりに特に効果的です。

### 特徴
- 最後のAmに向かって「落ちる」ような感覚がある
- 繰り返すと自然とクライマックス感が出る

### おすすめの雰囲気
😊 元気 / 🌸 春 / サビ・クライマックス

---

## ④ 小室進行（vi → IV → I → V）

```
Am → F → C → G
```

### 雰囲気
切なく・疾走感があり・エモーショナル。

### 使われる場面
90年代に小室哲哉さんが多用したことからこの名前がつきました。
現代のJ-POPでも広く使われています。

### 特徴
- マイナーから始まるので「暗い入り口」から始まり、Gで「抜け出す」感覚
- 繰り返すたびにどんどん感情が高まる

### おすすめの雰囲気
🌙 夜 / 切ない系 / 感動系バラード

---

## ⑤ 循環進行（I → vi → IV → V）

```
C → Am → F → G
```

### 雰囲気
懐かしく・穏やかで・安定した感じ。

### 使われる場面
1950〜60年代のドゥーワップやロックンロールで生まれた進行。
「スタンド・バイ・ミー」などが代表曲。

### 特徴
- 非常にシンプルで覚えやすい
- 「お決まり」の流れが聴き手を安心させる

### おすすめの雰囲気
😊 元気（レトロ系） / ノスタルジック / 懐かしい夏

---

## ⑥ 雨の進行（i → VII → VI → VII）

```
Dm → C → Bb → C
```

### 雰囲気
暗く・静かで・解決しない緊張感。

### 使われる場面
マイナーキーの楽曲のAメロ・イントロによく使われます。
「終わりのない雨」のような感覚を表現できます。

### 特徴
- 長調のコード（C、Bb）が混じることで「暗いだけでなく複雑な感情」になる
- ループしても解決しないため、聴き手を引き込む

### おすすめの雰囲気
🌧 雨 / 🌙 夜 / 内省的・哲学的な曲

---

## ⑦ スリーコード（I → IV → V → I）

```
C → F → G → C
```

### 雰囲気
シンプルで・力強く・飾り気のない素直さ。

### 使われる場面
ブルース・ロック・民謡・フォークの基本。
3つのコードだけで成立する最もシンプルな進行。

### 特徴
- 学ぶのに最適なシンプルさ
- 「最後にCに戻る」安心感が強い

### おすすめの雰囲気
😊 元気（シンプル系） / ロック / フォーク

---

## 進行の選び方まとめ

| 雰囲気 | おすすめ進行 |
|---|---|
| 明るく・ポップ | 王道進行、4536進行、循環進行 |
| 感動・ドラマチック | カノン進行、4536進行 |
| 切ない・夜っぽい | 小室進行、雨の進行 |
| 悲しい・雨 | 雨の進行、マイナー系進行 |
| 春・爽やか | 王道進行、カノン進行 |
| シンプルにしたい | スリーコード |

---

## 補足：メジャー（長調）とマイナー（短調）

- **メジャー（長調）**：明るい、楽しい、前向きな印象
- **マイナー（短調）**：暗い、切ない、落ち着いた印象

どちらが良い・悪いではなく、作りたい雰囲気に合わせて選びます。
ほとんどの曲は2つを混在させて複雑な感情を表現しています。
