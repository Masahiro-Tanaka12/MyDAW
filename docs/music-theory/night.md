# 🌙 夜の曲の音楽知識

## 雰囲気

静かで・少し哀愁があり・落ち着いた印象を与える曲。
深夜の街・一人の時間・しっとりした感情に寄り添う雰囲気。
夜景動画・LoFi BGM・センチメンタルなシーンに合う。

---

## おすすめ BPM 帯

| テンポ感 | BPM |
|---|---|
| ゆっくり（バラード） | 60〜75 |
| 夜らしいテンポ | 75〜90 |
| 少し動きがある夜 | 90〜100 |

基準：**BPM 78〜85** が「夜」の定番テンポ帯。

---

## おすすめキー

- **Aマイナー**（ラ・シ・ド…）：暗さの王道。最も「夜」らしいキー
- **Dマイナー**：Aマイナーより少し重く、深みがある
- **Eマイナー**：哀愁があり、バラードに向く

---

## おすすめコード進行

### ① i → V → VII → IV（夜の定番）
```
Am → E → G → F
```
マイナーの暗さと、長調コードの光が交差する。
映画音楽・J-POPバラードでよく使われる。

### ② i → VI → III → VII
```
Am → F → C → G
```
「小室進行」とも呼ばれる。切なさと疾走感を両立。
J-POPで最も多用されるマイナー進行のひとつ。

### ③ i → iv → i → V（静かな夜）
```
Am → Dm → Am → E
```
同じマイナーコードどうしで落ち着いた空間を作る。
変化が少なく、環境音楽的な使い方に向く。

---

## おすすめベースの特徴

- **長く伸ばす**：各コードのルート音を1〜2小節持続させる
- **低めのオクターブ**：A1〜E2 あたりの深みある音域
- **動かしすぎない**：余白が夜らしい空気を作る

### ベースラインの例（Am-E-G-F に対して）
```
A1（1小節）→ E2（1小節）→ G2（1小節）→ F2（1小節）
```

---

## おすすめドラムパターン

### 4つ打ち（控えめに）
```
キック：1拍・2拍・3拍・4拍（やや弱め）
```
4つ打ちでも velocity を抑えることで夜らしい静けさが出る。

### 強弱のコツ
- 1拍目・3拍目：少し強め（velocity 0.8）
- 2拍目・4拍目：弱め（velocity 0.7）

### より夜らしくするなら（Phase5以降）
- ハーフタイムフィール（キックを2拍・4拍のみ）
- ブラシスネア（金属的でなく柔らかい音）

---

## 避けたい特徴

| 避けるもの | 理由 |
|---|---|
| メジャーキー（長調）を主体にする | 明るすぎて夜らしさが消える |
| BPM 110 以上 | 落ち着きが失われる |
| 強すぎるキック（velocity 0.9 以上） | 激しくなりすぎる |
| 細かく動くベース | 落ち着きが失われる |
| 明るい長調コードの連続 | 夜らしさが出ない |
