# 🌧 雨の曲の音楽知識

## 雰囲気

しとやかで・内省的で・少し憂鬱な印象を与える曲。
雨音のような静けさ・感傷・孤独感を音で表現する。
雨の日の風景動画・詩的なVlog・感動シーンのBGMに合う。

---

## おすすめ BPM 帯

| テンポ感 | BPM |
|---|---|
| とてもゆっくり | 55〜65 |
| 雨らしいテンポ | 65〜78 |
| 少し動きがある雨 | 78〜88 |

基準：**BPM 68〜72** が「雨」の最も雰囲気が出るテンポ帯。

---

## おすすめキー

- **Dマイナー**：「雨」の代名詞的なキー。重く・しっとりした音
- **Bマイナー**：より暗く・沈んだ印象
- **Fマイナー**：映画的な悲しさ

---

## おすすめコード進行

### ① i → VII → VI → VII（雨の定番）
```
Dm → C → Bb → C
```
暗いまま変化する進行。解決しない緊張感が「雨」らしさを生む。

### ② i → iv → VI → III
```
Dm → Gm → Bb → F
```
マイナー系のコードだけで構成。どこまでも暗く・静かな雰囲気。

### ③ i → VII → VI → V（終わりに向かう感じ）
```
Dm → C → Bb → A
```
最後のA（長調の5番目）が少し光をもたらし、複雑な感情を表現。

---

## おすすめベースの特徴

- **とても長く伸ばす**：各コードのルート音を1〜2小節 保持
- **低いオクターブ**：D2〜A1 の重みある音域
- **弱めの音量**：velocity 0.6〜0.7 で控えめに

### ベースラインの例（Dm-Am-F-C に対して）
```
D2（1小節）→ A1（1小節）→ F2（1小節）→ C2（1小節）
```

---

## おすすめドラムパターン

### 控えめな4つ打ち
```
キック：1拍・2拍・3拍・4拍（かなり弱め）
```
リズムをほとんど感じさせないくらい弱くすることで、
「雨音のような背景」として機能させる。

### 強弱のコツ
- 1拍目・3拍目：弱め（velocity 0.8）
- 2拍目・4拍目：かなり弱め（velocity 0.6）

### より雨らしくするなら（Phase5以降）
- ハーフタイムフィール（キックを2小節に1回）
- ライドシンバル（チリチリした高音が雨粒のよう）
- スネアなし・ブラシだけ

---

## 避けたい特徴

| 避けるもの | 理由 |
|---|---|
| BPM 90 以上 | 雨の静けさが失われる |
| 強いキック（velocity 0.9） | 激しくなりすぎる |
| メジャーキー主体 | 明るさが「雨」の世界観を壊す |
| 跳ねるベースライン | 軽快になりすぎて雰囲気が合わない |
| コードの動きが激しい | 落ち着きが失われる |
